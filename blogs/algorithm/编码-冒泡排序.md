## 代码

```php
<?php
function bubbleSort(array $arr)
{
    $counts = count($arr);
    if ($counts <= 0) return;
    for ($i = 0; $i < $counts; $i++) {
        for ($j = $counts - 1; $j > $i; $j--) {
            if ($arr[$j] < $arr[$j - 1]) {
                $tmp = $arr[$j];
                $arr[$j] = $arr[$j - 1];
                $arr[$j - 1] = $tmp;
            }
        }
    }
    return $arr;
}

echo "冒泡排序-正序" . "\n";
$arr = array(1, 12, 16, 7, 9);
$results = bubbleSort($arr);
print_r($results);
```
