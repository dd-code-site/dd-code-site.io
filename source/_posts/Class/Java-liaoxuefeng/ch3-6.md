字符类型`char`表示一个字符。Java的`char`类型除了可表示标准的ASCII外，还可以表示一个Unicode字符：

```java
package com.study.basics.liaoxuefeng;

/**
 * 数据类型的定义
 *
 * @author liudandan
 */
public class DataType {
    public static void main(String[] args) {
        //TODO:: 字符型
        char a = 'A';
        char zh = '中';
    }
}
```

注意`char`类型使用单引号`'`，且仅有一个字符，要和双引号`"`的字符串类型区分开。
